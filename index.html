Glorious Pattern Matcher

In the left box, type a series of characters representing an ordered pattern. In the right box, type a series of words separated by spaces. The test button will let you know if the words match the pattern specified in the first box.

Examples

Box 1: AAA

Box 2: Ho Ho Ho

Result: Match

Box 1: AAA

Box 2: Oh No No

Result: No match

Box 1: ABBA

Box 2: Knowing Me Knowing You

Result: No match

Box 1: ABBA

Box 2: Knowing Me Me Knowing

Result: Match

Box 1: ACDC

Box 2: I'mT.N.T.

Result: Match
